def trapezium(a, b, n, f):
    h = (b - a) / n
    f_values = f(np.linspace(a, b, n + 1))
    return h / 2 * np.sum(f_values[:-1] + f_values[1:])

        def trapezium_adaptive(a, b, n, f, p):
    i1 = trapezium(a, b, n, f)
    error = 1e10

    while error > p:
        n *= 2
        h = (b - a) / n
        i2 = i1 / 2 + h * np.sum(f(a + h * np.arange(1, n, 2)))

def simpson(a, b, N, f):
		h = (b - a) / N
		f_values = f(np.linspace(a, b, N + 1))
		return h / 3 * np.sum(4 * f_values[1::2] + f_values[2::2] + f_values[:-1:2])

        def simpson_adaptive(a, b, n, f, p):
    s = lambda: (1 / 3) * (f(a) + f(b) + 2 * np.sum(f(a + h * np.arange(2, n-1, 2))))
    t = lambda: (2 / 3) * np.sum(f(a + h * np.arange(1, n, 2)))

    h = (b - a) / n
    s1 = s()
    t1 = t()
    i1 = h * (s1 + 2 * t1)
    error = 1e10

    while abs(error) > p:
        n *= 2
        h = (b - a) / n
        s2 = s1 + t1
        t2 = t()
        i2 = h * (s2 + 2 * t2)
        error = abs(i2 - i1) / 15
        i1, s1, t1 = i2, s2, t2
    return i2

  • Método de Romberg
    • 🐱‍👤 Matemática:

      Cálculo de um termo R i , m+1R_{\ i \ , \ m+1}R i , m+1​﻿:

      • iii﻿ refere-se ao número de vezes que fizemos n=2∗nn = 2*n n=2∗n﻿. Por exemplo, i=1i=1i=1﻿ refere-se ao elemento com que inicializamos nnn﻿ e i=2i=2i=2﻿ refere-se aos elementos calculados com n=2∗nn = 2*n n=2∗n﻿.
      • mmm﻿ refere-se à ordem do erro. Por exemplo, m=1m =1m=1﻿ refere-se a um erro de 2ª ordem, h2h^2h2﻿, enquanto que m=2m = 2m=2﻿ refere-se a um erro de 4ª ordem, h4h^4h4﻿.

      Cálculo do erro de R i, mR_{\ i, \ m}R i, m​﻿:

      Procedimento:

      1. Calcular R1,1=InR_{1,1} = I_nR1,1​=In​﻿ e R2,1=I2nR_{2,1} = I_{2n}R2,1​=I2n​﻿ para um dado nnn﻿ inicial.
      1. Calcular R2,2R_{2,2}R2,2​﻿ a partir da fórmula em cima apresentada.
      1. Calcular R3,3=I4nR_{3,3} = I_{4n}R3,3​=I4n​﻿ e proceder de forma idêntica para calcular R3,2R_{3,2}R3,2​﻿ e R3,3R_{3,3}R3,3​﻿.
      1. A cada passo calcular o erro e verificar se atingimos a precisão desejada.
    • 🚨 Ter em atenção:
      • Este método pode dar convergências muito lentas para funções mal comportadas.
      • O erro obtido refere-se a R m, m−1R_{\ m, \ m-1}R m, m−1​﻿ e não a Rm, mR_{m, \ m}Rm, m​﻿. Na prática, o erro calculado é superior ao real.
    • 👨‍💻 Código:
      def romberg(a, b, n, f, p):
    r = np.zeros([50, 50])
    i = 0
    r[i, 0] = trapezium(a, b, n, f)
    error = 1e10

    while abs(error) > p:
        i += 1
        n *= 2
        h = (b - a) / n
        r[i, 0] = r[i - 1, 0] / 2 + h * np.sum(f(a + h * np.arange(1, n, 2)))

        for m in range(i):
            error = (r[i, m] - r[i-1, m]) / (4 ** (m + 1) - 1)
            r[i, m + 1] = r[i, m] + error

    return r[i, i]

      def gauss(a, b, n, f):
    x, w = np.polynomial.legendre.leggauss(n)
    x = (b - a) / 2 * x + (b + a) / 2
    w = (b - a) / 2 * w
    return np.sum(w * f(x))

    Considere-se o integral do tipo:

    A resolução é feita em 2 passos (considere-se o método da quadratura de gauss por exemplo):

    • Primeiro, fixamos o y e calculamos o integral variando o x. O resultado deste resultado pode ser visto como uma função que depende apenas de y.
    • Depois calculamos o integral do que sair do anterior normalmente, tal como se fosse uma função de 1 variável.
    • 🚨 Ter em atenção:

      # Exemplo simples a utilizar 2 vezes a regra do trapézio num domínio quadrado
def trapezium_2d(xa, xb, nx, ya, yb, ny, f):
    fy = [trapezium(xa, xb, nx, y, f) for y in np.linspace(ya, yb, ny+1)]
    h = (yb - ya) / ny
    return h / 2 * np.sum(fy[:-1] + fy[1:])
